# swift2

Тестовое задание выполнено полностью.

Функционал: 
  - по открытию приложения загружаются посты по ссылке.
  - присутствует сортировка по:
        - больше/меньше лайков
        - ранее/позже выставлен пост
  - по нажатию на "Show more" показывается весь текст поста и на её месте появляется кнопка "Show less" - уменьшает текст до 2 строк
  - по нажатию на пост открывается в новом окне весь пост
